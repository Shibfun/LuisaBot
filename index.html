Welcome to LuisaSwap Bot!

Hey there! @luisaiagent_bot lets you trade tokens, deploy meme coins, and manage your wallet on Shibarium. Your username, Loading..., gets you started!

Get Started

1. Chat with @luisaiagent_bot in Telegram.

2. Import your wallet private key via the bot's interface.

3. Use commands like /buys, /sells, or @username swap to trade!

Features

Wallet Management

- Wallet Import: Import your private key using the bot's settings.

- Check Balance: Use /balance to see your BONE and tokens.

Trading

- Buy Tokens: Use /buys BONE

(e.g., /buys 1 BONE 0x...).

- Sell Tokens: Use /sells

or /sells %
(e.g., /sells 100 0x... or /sells 50% 0x...).

- Swap User's Token: Buy someone's Low Meme Coin with @username swap BONE or High Meme Coin with @username swaps BONE.

- Sell User's Token: Sell someone's Low Meme Coin with @username sell % or High Meme Coin with @username sells % (100% retains 0.0001 tokens).

Meme Coins

- Low Meme Coin: Deploy with /deploylow Name Symbol (e.g., /deploylow MyCoin MC). Bot auto-adds liquidity.

- High Meme Coin: Deploy with /deployhigh Name Symbol (e.g., /deployhigh BigCoin BC). Bot auto-adds liquidity.

- Claim Fees: Get fees with /claimfeeslow or /claimfeeshigh.

Profile

- User Info: Check stats with /userinfo.

Token Management

- Add Token: Track tokens with /addtoken

.

Tips

- Verify token addresses before trading!

- Import your private key securely in the bot's settings.

- Use /start in Telegram for quick help.

- Contact admins via @luisaiagent_bot if stuck!
